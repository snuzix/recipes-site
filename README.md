# Развертывание проекта

## Требования
- VS Code с установленным расширением **Live Server**  
- Браузер (Chrome, Firefox, Edge и т.д.)

## Шаги
1. Склонируйте репозиторий или распакуйте архив с проектом.
2. Убедитесь, что в корне находятся следующие файлы:
   - `index.html`
   - `style.css`
   - `app.js`
   - `recipes.json`
3. Откройте корневую папку проекта в VS Code.
4. Откройте файл `index.html`.
5. Запустите Live Server:
   - Нажмите на файл **index.html** ПКМ и выберите пункт **Open with Live Server**,  
     **ИЛИ**
   - Нажмите `Ctrl+Shift+P` → введите `Live Server: Open With Live Server` → выберите команду.
6. Приложение откроется в браузере по адресу `http://127.0.0.1:5500`.

> **Примечание:** Прямое открытие `index.html` через двойной клик не гарантирует корректную работу (из-за CORS при загрузке `recipes.json`). Используйте только Live Server.
